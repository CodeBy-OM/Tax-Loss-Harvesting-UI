# Tax Loss Harvesting UI

## Project Overview

This project is a React-based UI for Tax Loss Harvesting, designed to help users visualize and manage tax-saving investment strategies efficiently.

Live Demo - https://tax-loss-harvesting-ui.vercel.app/
---

## Tech Stack

* React.js
* JavaScript
* CSS / Tailwind
* REST APIs

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/CodeBy-OM/Tax-Loss-Harvesting-UI.git
cd Tax-Loss-Harvesting-UI
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm start
```

### 4. Build for production

```bash
npm run build
```
